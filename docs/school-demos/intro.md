# Getting Started

## Required Components

This tutorial assumes you have the following as a minimum in order to complete all of the basic tutorials.

- A breadboard
- A ESP32-C3 supermini board
- 3 resistors of either 220 or 330 &Omega;
- Jumper wires of various lengths to connect things together
- A thermoresistor
- A potentiometer
- A photoresistor
- LED's of various colours
- a RGB LED
- A power supply module (this is optional, the ESP32 can provide power on its own as well)
- A RFID-RC522 reader (or similar)

## Software

You will need to download and install [Visual Studio Code](https://code.visualstudio.com/download) from the official site onto a laptop or desktop PC

Once you have this ready you can proceed with the rest of the tutorial, we will go through every step needed in order to get everything working for you.
